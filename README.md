# Resume–Job Description Matching Agent

## Overview

This project uses a Siamese BiLSTM Neural Network to determine whether a candidate's resume matches a given job description.

## Features

- Resume preprocessing
- Job Description preprocessing
- Siamese BiLSTM model
- Resume–JD matching
- Skill gap analysis
- Experience display
- Model evaluation

## Technologies Used

- Python
- TensorFlow
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Google Colab

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Open the notebook in Google Colab or Jupyter Notebook.

3. Run all cells sequentially.

## Dataset

The dataset is not included in this repository due to its large size.
